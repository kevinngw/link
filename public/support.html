Get Help

If you find a bug, have a feature request, or need help with Link Pulse, use one of the support channels below.

Contact

  • Email the developer at mingnanwu@gmail.com
  • Open a support request on GitHub
  • View the project source
  • Read the privacy policy

What to include

  • Your device model and iOS version
  • The app version you are using
  • The line, stop, or station involved
  • Screenshots or screen recordings when possible

Support scope

Link Pulse is an independent transit board app for Link, RapidRide, and Swift. Support covers app behavior, rendering issues, favorites, nearby search, sharing, and realtime data presentation.

Realtime data quality also depends on upstream public transit feeds. If a problem looks tied to source data, please include the station name, direction, and approximate time so it can be traced more quickly.
